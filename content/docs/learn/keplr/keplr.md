Keplr is one of the commonly used Cosmos wallets for Terra Classic. Use this section to install Keplr, create or import accounts, send assets, stake, vote, and test on `rebel-2`.

## Before using Keplr with Terra Classic

- Confirm the active network is Terra Classic mainnet (`columbus-5`) or the intended testnet.
- Confirm your account address starts with `terra`.
- Keep seed phrases offline and never enter them into websites.
- Use a small test transaction before sending meaningful value.
- Keep enough LUNC liquid for fees.
- Verify exchange memo requirements before deposits.

## Safety baseline

Keplr should ask you to connect, approve, or sign. It should not ask you to reveal your recovery phrase during normal website use.

If a website asks for your seed phrase, close it.
